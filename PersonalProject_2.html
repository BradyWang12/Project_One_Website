Android Studio Project

Connect Four in Android Studio

Project Description

Developed a fully functional Connect Four game on Android featuring turn-based gameplay and a user-friendly interface. Players take alternate turns, trying to connect four pieces in a row. Key mechanics implemented include:

  • Turn-based system for two players
  • Win detection: horizontal, vertical, and diagonal
  • Game reset system for continuous play
  • Interactive UI with Android Studio components
  • Visual clarity and accessibility for ease of use

Skills learned from completing the project

  • Android Studio development and UI design
  • Implemented game logic in Java
  • Turn-based system and win condition algorithms
  • Debugging and bug testing to ensure correct gameplay
  • Project planning and iterative development workflow
